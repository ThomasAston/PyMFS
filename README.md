## Setup
To run this project, enter

```
$ cd ../PyMFS
$ python3 PyMFS.py
```

The file that is run and the materials which are used are currently set in PyMFS.py by the following lines

```
# Input file and materials file
filename = 'Tinput.tro'
material = 'materials.json'
```
